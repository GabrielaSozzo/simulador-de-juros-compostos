<P>
  Este é um projeto simples de uma calculadora de juros compostos desenvolvido utilizando HTML, CSS e JavaScript. A aplicação permite inserir o valor inicial, a taxa de juros mensal e o período em meses, realizando automaticamente o cálculo do montante final com base na fórmula de juros compostos.

O objetivo do projeto foi praticar conceitos fundamentais de desenvolvimento web, como estruturação com HTML, estilização com CSS e manipulação do DOM e lógica matemática com JavaScript.
</P>
